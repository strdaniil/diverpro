Empowering Swimmers Through Data

DiverPro is a computer-vision-based tool that helps swimmers analyze and perfect their dive starts. Using pose estimation and physics-based trajectory modeling, it allows for clear visual insights and improvement.

Technology

The project uses Python, Numpy, OpenCV,and MediaPipe to process and visualize dive data.
